- Alerts, Security, Memory Hit Rate, Durable Facts tiles should have flat horizontal bands

- If any metric is in danger range, that tile should have red tint
- Sessions, Alerts, Security tiles should use their accent color (blue, red) regardless

**Live morph (D-04):**
- Watch HeroStatsBar for 10+ seconds — as data polls every 5s, sparkline curves should smoothly transition to new shapes (visible on Sessions/Advisor Savings tiles with real data)

**Layer ordering (D-02):**
- Sparkline should be behind the left-edge radial glow (Phase 03)
- Both should be behind the text content
- Text should be fully readable

- Tiles should have rounded corners and subtle border
- Tiles should be taller than before (~72px minimum)
- Grid layout should be 7 columns on desktop, 4 on tablet, 2 on mobile
- Outer container should be unchanged (same card background)
