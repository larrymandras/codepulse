**Part B: Create CollapsibleSection inline component in Capabilities.tsx (D-06, D-07)**

Add a `CollapsibleSection` component inside `Capabilities.tsx` (above the default export, after the existing `HooksPanel` component). This is a page-local component since it's specific to the Capabilities page layout.

```typescript
function CollapsibleSection({
  title,
  children,
  defaultExpanded = true,
  count,
}: {
  title: string;
  children: React.ReactNode;
  defaultExpanded?: boolean;
  count?: number;
}) {
  const [expanded, setExpanded] = useState(defaultExpanded);
  return (
    <div>
      <button
        onClick={() => setExpanded((prev) => !prev)}
        className="flex items-center gap-2 w-full text-left mb-2 group"
      >
        <span className="text-sm font-semibold uppercase tracking-wide text-muted-foreground">
          {title}
        </span>
        {count !== undefined && (
          <span className="text-xs text-muted-foreground opacity-50">{count}</span>
        )}
        <span className="ml-auto text-muted-foreground">
          {expanded ? (
            <ChevronUp className="w-3 h-3" />
          ) : (
            <ChevronDown className="w-3 h-3" />
          )}
        </span>
      </button>
      {expanded && children}
    </div>
  );
}
```

State is React `useState` only -- session-only per D-07. When user navigates away and returns, all sections default to expanded.

Add `ChevronDown` and `ChevronUp` to the existing lucide-react import line at the top of the file:

Create a `useMemo` inside the `Capabilities` component that builds the placeholder from the panel names that currently have data:

```typescript
const searchPlaceholder = useMemo(() => {
  const panels: string[] = [];
  if (servers && servers.length > 0) panels.push("servers");
  if (plugins && plugins.length > 0) panels.push("plugins");
  if (skills && skills.length > 0) panels.push("skills");
  if (hooks && hooks.length > 0) panels.push("hooks");
  if (tools && tools.length > 0) panels.push("tools");
  if (catalogCommands.length > 0) panels.push("commands");
  if (panels.length === 0) return "Search...";
  return `Search ${panels.join(", ")}...`;
}, [servers, plugins, skills, hooks, tools, catalogCommands]);
```

Use `searchPlaceholder` in the `<input>` element's `placeholder` prop instead of the hardcoded string:
