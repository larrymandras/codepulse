**Step 1 — Create `src/hooks/useNavCounts.ts`** (per Pitfall 5 — single subscription, not 15):

This hook returns all sidebar badge counts from a SINGLE Convex query to avoid re-render storms.

```typescript
import { useQuery } from "convex/react";
import { api } from "../../convex/_generated/api";

interface NavCounts {
  activeSessions: number;
  activeAgents: number;
  unreadAlerts: number;
}

export function useNavCounts(): NavCounts {
  // Attempt to use an existing aggregated query, or fall back to individual queries
  // Check convex/ for existing count queries and use them
  // If no aggregated query exists, create individual useQuery calls but memoize the result object
  const sessions = useQuery(api.sessions.listActive);
  const agents = useQuery(api.agents.list);
  const alerts = useQuery(api.alerts.list);

  return {
    activeSessions: sessions?.length ?? 0,
    activeAgents: agents?.filter((a: any) => a.status === "active")?.length ?? 0,
    unreadAlerts: alerts?.filter((a: any) => !a.acknowledged)?.length ?? 0,
  };
}
```

IMPORTANT: Read `convex/schema.ts` and the actual query exports in `convex/sessions.ts`, `convex/agents.ts`, `convex/alerts.ts` to determine the correct API paths and filter logic. The above is a template — adapt to the actual Convex API surface. If there are no appropriate queries, create simple count queries in Convex or use the closest existing queries.
