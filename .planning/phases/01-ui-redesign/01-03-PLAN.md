**Step 1 — Create `src/hooks/useNavCounts.ts`** (per D-10, Pitfall 5 — single subscription pattern, not 15):

This hook returns all sidebar badge counts. Per D-10 "live count badges on every nav item", the hook must cover ALL nav items that have countable Convex data. Items without meaningful counts (Analytics, Infrastructure, Self-Healing, Profiles, Settings) get no badge — this is acknowledged, not a scope reduction.

```typescript
import { useQuery } from "convex/react";
import { api } from "../../convex/_generated/api";

interface NavCounts {
  activeSessions: number;
  activeAgents: number;
  recentExecutions: number;
  activeBuildJobs: number;
  automationJobs: number;
  securityEvents: number;
  memoryEntries: number;
  capabilities: number;
  briefings: number;
  unreadAlerts: number;
}

export function useNavCounts(): NavCounts {
  // Read each Convex query file to find the correct exported query names.
  // Use list/count queries and derive counts. Adapt API paths to actual exports.
  const sessions = useQuery(api.sessions.listActive);
  const agents = useQuery(api.agents.listAll);
  const executions = useQuery(api.commandExecutions.list);
  const buildJobs = useQuery(api.build.list);
  const automation = useQuery(api.automation.list);
  const security = useQuery(api.security.list);
  const memory = useQuery(api.memory.list);
  const capabilities = useQuery(api.registry.list);
  const briefings = useQuery(api.reflections.list);
  const alerts = useQuery(api.alerts.list);

  return {
    activeSessions: sessions?.length ?? 0,
    activeAgents: agents?.filter((a: any) => a.status === "active")?.length ?? 0,
    recentExecutions: executions?.length ?? 0,
    activeBuildJobs: buildJobs?.filter((b: any) => b.status === "running")?.length ?? 0,
    automationJobs: automation?.length ?? 0,
    securityEvents: security?.length ?? 0,
    memoryEntries: memory?.length ?? 0,
    capabilities: capabilities?.length ?? 0,
    briefings: briefings?.length ?? 0,
    unreadAlerts: alerts?.filter((a: any) => !a.acknowledged)?.length ?? 0,
  };
}
```

IMPORTANT: The above is a TEMPLATE. Read EACH Convex query file referenced in read_first to determine:

2. The correct filter logic (e.g., active agents filter, unread alerts filter)
3. If a query doesn't exist, check if there's a close alternative or use the most reasonable list query

If a Convex module has no appropriate list query for counting, either:
