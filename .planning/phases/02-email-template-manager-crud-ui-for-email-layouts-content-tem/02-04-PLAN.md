**Step 3 — Create `src/components/email/EmailPreviewPane.tsx`:**

Props:
```typescript
interface EmailPreviewPaneProps {
  slug: string;
  html: string;
  subject: string;
  variables: Record<string, VariableDefinition>;
  disabled?: boolean; // true in create mode
}
```

Implementation:
- **Create mode (`disabled=true`):** Show static placeholder centered: "Save the template first to enable preview." (text-sm text-muted-foreground, bg-muted/20 rounded, centered vertically and horizontally)
- **Edit mode:** Debounced preview with 500ms delay
  - State: `previewHtml`, `previewLoading`, `previewError`, `channel` ("smtp" | "gmail")
  - **Channel toggle:** shadcn `ToggleGroup` (type="single") with "SMTP" and "Gmail" pills. Default: "smtp". On change: triggers debounced re-fetch with new `channel` value.
  - **Debounce pattern (per RESEARCH.md Pattern 6):**
    ```typescript
    const timerRef = useRef<ReturnType<typeof setTimeout>>(undefined);
    useEffect(() => {
      clearTimeout(timerRef.current);
      timerRef.current = setTimeout(async () => {
        setPreviewLoading(true);
        setPreviewError(false);
        try {
          const sampleVars = buildSampleVariables(variables);
          const result = await previewTemplate(slug, { variables: sampleVars, channel });
          setPreviewHtml(result.html);
        } catch {
          setPreviewError(true);
        } finally {
          setPreviewLoading(false);
        }
      }, 500);
      return () => clearTimeout(timerRef.current);
    }, [slug, html, subject, variables, channel]);
    ```
  - **Preview header:** `flex items-center justify-between px-3 py-2 border-b border-border bg-muted/20 text-xs`
    - Left: "Preview" label (text-muted-foreground)
    - Center: channel ToggleGroup
    - Right: loading spinner (`Loader2 h-3 w-3 animate-spin`) or "Updating preview..." text when loading
  - **Preview body:**
    - Error state: "Preview failed. Check your template syntax and variable definitions." (text-sm text-destructive, centered in pane)
    - Success: `<iframe srcDoc={previewHtml} sandbox="allow-same-origin" className="flex-1 w-full border-0" title="Email preview" />`
    - CRITICAL SECURITY: iframe uses `srcDoc` NOT `src`. Sandbox does NOT include `allow-scripts`. This prevents any `<script>` tags in template HTML from executing.
  - **Container:** `flex flex-col h-full border border-border rounded-md overflow-hidden`

| T-02-12 | Denial of Service | EmailPreviewPane debounce | mitigate | 500ms debounce prevents rapid-fire API calls on every keystroke. Timer cleanup on unmount prevents stale state updates. |
</threat_model>
