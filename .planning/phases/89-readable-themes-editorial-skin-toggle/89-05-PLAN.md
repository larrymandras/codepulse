| localStorage (origin-shared) -> document.documentElement attribute | Another script on the same origin can write the codepulse-theme value; the inline script consumes it pre-paint |
| index.html inline script -> DOM | Synchronous script sets an attribute before React loads |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-89-09 | Tampering | localStorage codepulse-theme value read by inline script | mitigate | Validate the value against the 4-slug allowlist (cyan/emerald/readable/aubergine) BEFORE setAttribute; fall back to cyan on any invalid/absent value (RESEARCH script) |
| T-89-10 | Injection (XSS) | inline pre-paint script | mitigate | Never interpolate the raw localStorage value into innerHTML/eval; only pass a validated allowlisted slug to setAttribute('data-theme', t). Static author-controlled script, no dynamic code |
| T-89-11 | Tampering | inline script under future CSP | accept | No CSP exists today; code comment flags that a nonce/'unsafe-inline' would be required if CSP is added later. Out of scope for this phase |
</threat_model>
