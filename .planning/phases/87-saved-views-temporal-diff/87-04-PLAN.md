    Frame fetch effect (Pitfall 4): keyed on `[currentFrameIndex, frames]`. `const key = frames[currentFrameIndex];`

    ahead by issuing cache-miss fetches for `frames[currentFrameIndex+1]` and `+2` (cache-checked, fire-and-forget).

    Playback timer: `useEffect` keyed on `[isPlaying, fps, frames.length]` running
    `setInterval(() => setCurrentFrameIndex(i => Math.min(i+1, frames.length-1)), 1000/fps)` with clearInterval
    cleanup. Stop advancing (or set isPlaying false) at the last frame.

    State + API: `isPlaying`, `fps` (default 1), `currentFrameIndex`, `currentGraph: KgGraphData | null`,
