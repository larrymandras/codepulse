**Implementation details per D-01, D-02, D-03:**
- `memo` wrapped (export `const DialGauge = memo(DialGaugeInner)` then `export default DialGauge`)
- Import `{ memo, useId }` from "react", `{ motion, useReducedMotion }` from "motion/react"
- Import `{ thresholdTone, AnimatedNumber, ThresholdConfig }` from "./MetricCard"
- SVG: `viewBox="0 0 76 76"`, `width={76}`, `height={76}`
- Arc constants: `R = 28`, `CIRCUMFERENCE = 2 * Math.PI * R * (270 / 360)` (approximately 131.95)
- `arcPath270(cx, cy, r)` helper using trig with `Math.cos`/`Math.sin` (NO `**` operator, use `Math.PI / 180` for degree conversion):
  - Start angle: 135 degrees (bottom-left)
  - End angle: 45 degrees (bottom-right)  
  - Returns `M ${sx} ${sy} A ${r} ${r} 0 1 1 ${ex} ${ey}`
- Clamp pct: `Math.min(Math.max(value ?? 0, 0.5), 99.5)` to avoid degenerate arc
- Track path: full arc in `var(--border)` color, 8px stroke, strokeLinecap="round"
- Fill path: `motion.path` with `animate={{ strokeDashoffset: offset }}` where `offset = CIRCUMFERENCE * (1 - safePct / 100)`
  - `strokeDasharray={CIRCUMFERENCE}`
  - `stroke={var(--tone-${tone})}` where tone = `thresholdTone(value ?? 0, thresholds)`
  - `transition={shouldReduce ? { duration: 0 } : { duration: 0.5, ease: "easeOut" }}`
  - Glow: `filter: drop-shadow(0 0 4px color-mix(in oklch, var(--tone-${tone}) 40%, transparent))`
- Center text overlay: absolutely positioned div centered over SVG
  - Value: `AnimatedNumber value={value ?? 0} format={format ?? (v => Math.round(v) + '%')}`  at 20px/600 weight
  - Label: `label` at 12px/400 weight below value
- Empty state: when `value == null`, render gauge at 0% fill with center showing em dash `—` instead of AnimatedNumber, label still shown
- `useId()` with `.replace(/:/g, '')` for any SVG gradient/filter IDs
- `role="meter"`, `aria-valuenow={value ?? 0}`, `aria-valuemin={0}`, `aria-valuemax={100}`, `aria-label={label + ': ' + (value ?? 0) + '%'}`
