## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, gauge label-to-value gap, pill gutter |
| sm | 8px | Segment expand panel padding, tick label spacing |
| md | 16px | Gauge row padding, provider row padding, window bar padding |
| lg | 24px | Section-to-section gap on Analytics page |
| xl | 32px | Gauge row to model strip gap |
| 2xl | 48px | Analytics page section breaks |
| 3xl | 64px | Page-level top padding |

Exceptions:
- Dial gauge fixed size: 76px × 76px (dictated by D-02; not a spacing unit)
- Window bar tick labels: 4px gap above tick mark to label (xs)
- Segment expand panel: 12px vertical padding (3px non-standard — use `py-3` / 12px) — acceptable compromise for panel density
- Preset pill buttons: `h-7 px-3` (28px height, 12px horizontal) — matches existing shadcn pill sizing

**Accent reserved for (explicit list):**
- Gauge arc fill stroke — driven by `thresholdTone()`: `--tone-good`, `--tone-warn`, `--tone-danger`
- Gauge arc glow (`filter: drop-shadow` at 40% opacity of tone color)
- Model split strip segment gradient fills — sequential OKLCH palette (see below)
- Provider row radial gradient background — `--provider-{name}` token via `data-provider` attribute
- Window bar gradient fill — `--accent-cost` (amber) applied as linear-gradient from left to current position
- Preset pill button active state — `--primary` underline or background tint

**Gauge tone color mapping** (from Phase 04 `thresholdTone()` + existing `--tone-*` tokens):
- `good` → `--tone-good`: `oklch(0.72 0.17 142)` (green, dark mode)
- `warn` → `--tone-warn`: `oklch(0.78 0.15 80)` (amber, dark mode)
- `danger` → `--tone-danger`: `oklch(0.72 0.20 27)` (red, dark mode)

## Component Inventory

### New Components (Phase 05)

#### `DialGauge`
- Location: `src/components/DialGauge.tsx`
- Size: 76px × 76px SVG, viewBox `0 0 76 76`
- Arc geometry: 270-degree arc with 90-degree bottom gap (bottom-center gap). Start: 135° from top-right, end: 45° from top-left (clockwise fill).
- SVG path: Two overlapping `<circle>` elements using `strokeDasharray`/`strokeDashoffset` on a clipped arc, or computed arc path. Use `strokeDashoffset` animation (not CSS transition on `d` attribute).
- Track stroke: 8px, `--secondary` color, `strokeLinecap="round"`
- Fill stroke: 8px, tone-driven color (`var(--tone-good/warn/danger)`), `strokeLinecap="round"`
- Glow: `filter: drop-shadow(0 0 4px color-mix(in oklch, var(--tile-tone) 40%, transparent))`
- Center text: value at 20px/600 weight, label at 12px/400 weight below it — vertically centered in the inner circle
- Animation: `strokeDashoffset` animates via Motion `animate` prop (spring, duration 600ms) on Convex subscription update
- Props: `value: number` (0–100), `label: string`, `thresholds: ThresholdConfig`
- Empty state: render gauge at 0% with `--muted` track, center shows `—`
- `prefers-reduced-motion`: skip animation, render at final value instantly

#### `GaugeRow`
- Location: `src/components/GaugeRow.tsx`
- Layout: `flex flex-row gap-6 justify-start` containing three `DialGauge` instances
- Gauges: Budget Utilization, Context Pressure, Rate Limit Proximity
- Labels: "Budget" / "Context" / "Rate Limit"
- Card wrapper: `data-accent="cost"` (uses existing Phase 03 radial gradient selector — cost accent covers all three budget/utilization metrics)
- Section heading: 16px/600 "Usage Gauges" above the row

#### `ModelSplitStrip`
- Location: `src/components/ModelSplitStrip.tsx`
- Container: full-width `h-6` (24px) horizontal stacked bar, `rounded-sm` ends
- Segments: `flex flex-row` of proportional-width `div` elements. Width = `(model.cost / totalCost) * 100 + '%'`
- Segment gradient: `linear-gradient(90deg, {model-color} 0%, color-mix(in oklch, {model-color} 70%, transparent) 100%)`
- Interaction: click segment → AnimatePresence slide-down panel below strip. Height animates 0 → auto. Only one panel open at a time.
- Expand panel layout: `grid grid-cols-2 gap-3 p-3` — model name, total cost, percentage, input tokens, output tokens
- Segment active state: 2px solid white outline on active segment, others dim to 60% opacity
- Tooltip on hover (not click): model name in popover above segment, 400ms delay
- Min segment width: 8px — models below threshold get grouped as "Other"
- Section heading: "Model Distribution" at 16px/600

#### `ProviderRow`
- Location: `src/components/ProviderRow.tsx`
- Layout: `flex items-center gap-4 p-4 rounded-md` per row
- `data-provider="{name}"` attribute drives radial gradient background
- Left: provider name at 16px/600, model count label at 14px/400 below
- Right: cost value at 16px/600 tabular-nums, percentage at 14px/400 muted
- Stack: vertical list of `ProviderRow` items in a `GlassPanel` or card wrapper

#### `WindowBar`
- Location: `src/components/WindowBar.tsx`
- Container: full-width, `h-16` (64px) SVG or div-based bar
- Time range preset pills: `1h | 6h | 24h | 7d | 30d` — shadcn `Toggle` or custom pill row, default `24h` active
- Gradient fill: `linear-gradient(90deg, color-mix(in oklch, var(--accent-cost) 60%, transparent) 0%, var(--accent-cost) 100%)` from left up to current time position
- Tick marks: vertical lines at interval boundaries, `--border` color, 8px tall
- Tick labels: time strings at 12px/400 — "6a", "9a", "12p", "3p", "6p", "9p" for hourly; "Mon", "Tue", "Wed" for daily
- Section heading: "Usage Window" at 16px/600

## Interaction Contracts

### Dial Gauge Animation
- Trigger: Convex subscription delivers new utilization value
- Behavior: `strokeDashoffset` springs from current → new value over 600ms (Motion spring, stiffness 80, damping 20)
- No flash/jump: use `useMotionValue` initialized to last known value, not zero
- Stale state: if no update for >120s, reduce gauge opacity to 0.4 (matches ContextGauge pattern)

### Model Strip Expand
- Click segment → expand panel slides down with AnimatePresence (`initial={{ height: 0, opacity: 0 }}` → `animate={{ height: "auto", opacity: 1 }}` → `exit={{ height: 0, opacity: 0 }}`, duration 200ms ease-out)
- Click same segment again → close
- Click different segment → previous closes, new opens (no flicker)
- Keyboard: Enter/Space on focused segment toggles expand
- `aria-expanded` attribute on each segment button

### Window Bar Preset Pills
- Click pill → updates selected range, re-queries data, tick marks re-render
- Active pill: `bg-primary/15 text-primary border-primary/30` (matches tone pill pattern from Phase 04)
- Inactive pill: `bg-transparent text-muted-foreground border-border`

## SVG Geometry Reference

### 270° Dial Arc (for executor)
For a 76×76px SVG (center 38,38, radius 28):
- Track arc: 270° spanning from 135° to 45° (clockwise, skipping bottom 90°)
- Circumference of 270° arc: `2π × 28 × (270/360)` = `~131.9px`
- strokeDasharray: `{filled} {131.9}` where `filled = (value/100) × 131.9`
- Start angle (135° = bottom-left): `cx + r×cos(135°) = 38 - 19.8 = 18.2`, `cy + r×sin(135°) = 38 + 19.8 = 57.8`
- Arc path: Use `<circle>` with `strokeDashoffset` rotated via `transform="rotate(135, 38, 38)"` — simpler than computed `path d`
- Math: avoid `**` operator (esbuild constraint from D-01-04-01). Use `Math.pow()` or manual trig lookups.

## Accessibility

- All `DialGauge` elements: `role="meter"`, `aria-valuenow`, `aria-valuemin="0"`, `aria-valuemax="100"`, `aria-label="{label}: {value}%"`
- Model strip segments: `role="button"`, `aria-expanded`, `aria-label="{model name}: {cost} ({pct}%)"`, keyboard-focusable (`tabIndex={0}`)
- Preset pills: `role="radio"` group, `aria-checked` on active
- Tick marks: `aria-hidden="true"` (decorative)
- Window bar: `aria-label="Usage window: {range}"` on container
- `prefers-reduced-motion`: All Motion animations skip to final value instantly — enforce via existing global `@media (prefers-reduced-motion: reduce)` block in `index.css`
