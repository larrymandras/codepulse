SSE consumption effect — when `runId` changes and is non-null:
```typescript
useEffect(() => {
  if (!runId) return;
  // Reset state for new run
  setLogLines([]);
  setIframeContent("");
  setAccumulatedText("");
  setStreamStatus("streaming");
  setProgress(0);
  setErrorMessage(null);

  const controller = new AbortController();
  let textBuffer = "";

  streamRunEvents(runId, {
    onToken: (text) => {
      textBuffer += text;
      setAccumulatedText(textBuffer);
      setLogLines(prev => [...prev, text]);
      // Extract artifact HTML from accumulated text
      const artifact = extractArtifact(textBuffer);
      if (artifact) setIframeContent(artifact);
      // Approximate progress (rough heuristic based on text length)
      setProgress(prev => Math.min(prev + 0.5, 95));
    },
    onError: (err) => {
      setStreamStatus("error");
      setErrorMessage(err.message || "Generation failed. The daemon returned an error.");
    },
    onDone: () => {
      setStreamStatus("complete");
      setProgress(100);
      onGenerationComplete();
    },
    signal: controller.signal,
  });

  return () => controller.abort();
}, [runId]);
```

`extractArtifact` function (inline in module):
```typescript
function extractArtifact(text: string): string | null {
  const m = /<artifact[^>]*>([\s\S]*?)<\/artifact>/i.exec(text);
  return m ? m[1].trim() : null;
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Daemon SSE stream -> Browser | Untrusted streaming content from agent generation |
| Generated artifact HTML -> srcdoc iframe | Untrusted HTML rendered in sandboxed iframe |
| Export blob -> User download | File downloaded from daemon, consumed by user |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-11 | Tampering | StreamingPreview.tsx srcdoc | mitigate | iframe uses `sandbox="allow-scripts"` WITHOUT `allow-same-origin` — prevents artifact HTML from accessing CodePulse origin DOM, cookies, or localStorage |
| T-01-12 | Tampering | StreamingPreview.tsx extractArtifact | accept | Regex extracts content between artifact tags; malformed tags result in no preview (safe failure) |
| T-01-13 | Information Disclosure | ExportPanel.tsx blob download | accept | Blob is generated by local daemon from user's own project; no cross-user data exposure risk |
| T-01-14 | Denial of Service | StreamingPreview.tsx SSE | mitigate | AbortController cleanup on component unmount prevents orphaned streams; progress capped at 95% during streaming to avoid UI jank |
</threat_model>
