**SkillPicker.tsx** — Per UI-SPEC "Skill Picker (Step 1)":

Props:
```typescript
interface SkillPickerProps {
  selectedSkillId: string | null;
  onSelect: (skillId: string) => void;
}
```

Implementation:
- `useState` for `skills: Skill[]` (starts empty), `loading: boolean` (starts true), `error: string | null`, `search: string`.
- `useEffect` on mount: call `fetchSkills()` from `@/lib/openDesignApi`. On success -> set skills + loading false. On error -> set error message + loading false.
- Filter: `const filtered = skills.filter(s => s.title.toLowerCase().includes(search.toLowerCase()) || s.category.toLowerCase().includes(search.toLowerCase()))`.
- Search input: `<input type="text" placeholder={`Search ${skills.length} skills...`} value={search} onChange={e => setSearch(e.target.value)} className="w-full px-3 py-2 text-sm bg-card/60 border border-border/40 rounded-xl text-foreground placeholder:text-muted-foreground focus:outline-none focus:ring-1 focus:ring-primary/50" />`
- Grid: `<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4 mt-4">`
- Loading: 6 skeleton cards (reuse SkeletonCard pattern from CatalogBrowser — `bg-card/60 border border-border/40 rounded-xl p-4 animate-pulse`).
- Error: `<div className="flex items-center gap-2 p-3 rounded-lg bg-destructive/10 border border-destructive/30 text-destructive text-sm"><AlertCircle className="h-4 w-4 shrink-0" /><span>{error}</span></div>` (import AlertCircle from lucide-react).
- Each card: `<div onClick={() => onSelect(skill.id)} role="radio" aria-checked={selectedSkillId === skill.id} className={cn("bg-card/60 backdrop-blur-sm border rounded-xl p-4 flex flex-col gap-2 cursor-pointer transition-all", selectedSkillId === skill.id ? "border-primary/60 bg-primary/5" : "border-border/40 hover:border-primary/40")}>`
  - Title: `<h3 className="text-sm font-medium text-foreground">{skill.title}</h3>`
  - Category: `<span className="text-[10px] uppercase tracking-wider text-muted-foreground bg-muted/50 px-1.5 py-0.5 rounded w-fit">{skill.category}</span>`
  - Summary: `<p className="text-xs text-muted-foreground line-clamp-2">{skill.summary}</p>`
- Empty filtered: `<p className="text-center text-sm text-muted-foreground py-8">No skills match your search</p>`
- Import `cn` from `@/lib/utils`.

```tsx
<div className="flex items-center gap-1 px-6 pt-5 pb-3 border-b border-border/30">
  {STEP_LABELS.map((label, idx) => (
    <React.Fragment key={label}>
      {idx > 0 && <div className="flex-1 h-px bg-border" />}
      <button
        onClick={() => canGoToStep(idx) && setCurrentStep(idx)}
        disabled={!canGoToStep(idx)}
        aria-current={idx === currentStep ? "step" : undefined}
        aria-label={`Step ${idx + 1} of 6: ${label}`}
        className={cn(
          "flex items-center gap-1.5 px-2 py-1 text-xs rounded transition-colors",
          idx === currentStep ? "text-primary font-semibold" : "text-muted-foreground",
          idx < currentStep ? "text-muted-foreground" : "",
          !canGoToStep(idx) ? "opacity-50 cursor-not-allowed" : "cursor-pointer hover:text-foreground"
        )}
      >
        {idx < currentStep ? (
          <Check className="h-3.5 w-3.5" />
        ) : (
          <span className={cn(
            "w-5 h-5 rounded-full flex items-center justify-center text-[10px] font-medium border",
            idx === currentStep ? "border-primary bg-primary/10 text-primary" : "border-border text-muted-foreground"
          )}>{idx + 1}</span>
        )}
        <span className="hidden sm:inline">{label}</span>
      </button>
    </React.Fragment>
  ))}
</div>
```

Navigation helper:
```typescript
function canGoToStep(idx: number): boolean {
  // Can't go back past step 4 once generation has started
  if (generationStarted && idx < 4) return false;
  // Can't skip ahead past the current step
  if (idx > currentStep) return false;
  return true;
}
```

Step content area — renders based on `currentStep`:
- Step 0: `<SkillPicker selectedSkillId={selectedSkillId} onSelect={setSelectedSkillId} />`
- Step 1: `<DesignSystemPicker selectedDesignSystemId={selectedDesignSystemId} onSelect={setSelectedDesignSystemId} />`
- Step 2: `<DiscoveryForm brief={brief} onBriefChange={setBrief} onSubmit={() => setCurrentStep(3)} />`
- Step 3: `<div className="py-16 text-center text-muted-foreground text-sm">Direction picker — coming in next update</div>` (placeholder for Plan 04)
