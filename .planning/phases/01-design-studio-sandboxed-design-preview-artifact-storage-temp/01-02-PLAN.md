**IframeEmbed.tsx** — Per UI-SPEC IframeEmbed section:
- Props: none (reads `VITE_OPEN_DESIGN_URL` from `import.meta.env`).
- Internal state: `status: 'loading' | 'ready' | 'error'`. Default `'loading'`.
- `useEffect` on mount: poll `checkHealth()` every 2s. On first success → set `'ready'`. After 10s with no success → set `'error'`. On success, clear the polling interval.
- Render logic:
  - `status === 'loading'`: Show skeleton overlay. Use `<div className="absolute inset-0 bg-background/60 backdrop-blur-sm flex items-center justify-center z-10">` with `<Loader2 className="h-5 w-5 animate-spin" />` and `<span className="text-sm text-muted-foreground">Connecting to Design Studio...</span>`.
  - `status === 'error'`: Show error overlay with heading "Design Studio Unavailable" (text-base semibold), body "The Open Design daemon is not running. Start it with `docker compose up open-design` and refresh." (text-sm muted), and "Retry Connection" button (`bg-primary text-primary-foreground`) that resets to `'loading'` and restarts polling.
  - `status === 'ready'`: Show `<iframe src={import.meta.env.VITE_OPEN_DESIGN_URL} title="Design Studio" sandbox="allow-scripts allow-same-origin allow-forms" className="w-full border-0" style={{ height: 'calc(100vh - 56px)' }} />`. Apply opacity transition: 200ms ease-out from 0 to 1.
- The component container is `<div className="relative w-full" style={{ minHeight: 'calc(100vh - 56px)' }}>`.
- Import `Loader2` from `lucide-react`. Import `checkHealth` from `@/lib/openDesignApi`.
