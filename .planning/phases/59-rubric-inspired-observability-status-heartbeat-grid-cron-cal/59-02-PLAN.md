Constants:
```typescript
const IDLE_THRESHOLD_MS = 5 * 60 * 1000; // 5 minutes
```

Component logic:
1. **Convex data**: `const recentEvents = useRecentAgentStatus();`
2. **WS subscription for instant updates**: Use `subscribeEvent("agent_status", callback)` to update local state `liveStates: Record<string, { agentId: string; state: string; currentTask?: string; errorCount?: number; timestamp: number }>`
3. **Idle timeout via interval**: `const [now, setNow] = useState(Date.now());` + `useEffect` with `setInterval(() => setNow(Date.now()), 1000)` and cleanup
4. **Derive state function**:
   ```typescript
   function deriveState(event: { state: string; timestamp: number } | undefined, now: number): AgentState {
     if (!event) return "idle";
     if (now - event.timestamp * 1000 > IDLE_THRESHOLD_MS) return "idle";
     return event.state as AgentState;
   }
   ```
   (Note: timestamp from Convex is in seconds (Python `time.time()`), multiply by 1000 for JS milliseconds)
5. **Merge Convex + WS data**: For each agent in AGENT_ROSTER, find the latest event from either liveStates or recentEvents (group recentEvents by agentId, take latest). Derive state from merged event + now.
6. **Selected tile state**: `const [selectedAgentId, setSelectedAgentId] = useState<string | null>(null);`
7. **Click handler**: `onClick={() => setSelectedAgentId(prev => prev === agentId ? null : agentId)` (per D-04)
