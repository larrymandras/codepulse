| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-68-01 | Information Disclosure | convex/gatewayQuota.ts pollAndStore | mitigate | Never log `process.env.ASTRIDR_API_KEY`; log only response status codes |
| T-68-02 | Tampering | convex/gatewayTasks.ts upsert | mitigate | Convex `v.` validators enforce field types; `v.string()` for taskId prevents injection; `v.float64()` for durationSeconds rejects non-numeric |
| T-68-03 | Tampering | convex/routingDecisions.ts insert | mitigate | `v.optional(v.float64())` rejects non-numeric score values; `v.boolean()` for fallbackUsed prevents string "true" storage |
| T-68-04 | Denial of Service | convex/gatewayQuota.ts pollAndStore | accept | Cron runs every 5 min (fixed rate); even if gateway is slow, Convex action has built-in timeout; no amplification vector |
