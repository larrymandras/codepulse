## Interaction Contracts

### GatewayQuotaPanel
- Renders quota bars for API-billed providers only (claude-sdk, anthropic_direct, openrouter)
- Subscription providers (claude-cli, codex, antigravity) render a `UNLIMITED` badge (`bg-gray-700/50 text-gray-400`) — no bar
- Each bar shows: provider display name (left), percentage text + spend USD amount (right, tabular-nums mono)
- Bar track: `w-full h-1 bg-gray-700/50` (matches ProviderHealthPanel track)
- Bar fill: color-thresholded per D-06 color rules above
- Data source: `useQuery(api.gatewayQuota.latestByProvider)` — reactive, no manual refresh
- Empty state: "No quota data yet. Gateway quota polling begins on next cron cycle." (plain `text-sm text-muted-foreground`)
- Loading state: Skeleton bars (shadcn `Skeleton` component)

### RoutingDecisionsTable
- Default 25 rows, most recent first (descending timestamp)
- Columns: Task ID (8-char mono truncation with tooltip showing full ID), Requested Provider, Selected Provider, Fallback (`YES`/`NO` badge), Timestamp (relative: "3m ago")
- Row expansion: Click row to expand inline — shows score grid: Quota Score, Latency Score, Cost Score, Final Score (all `tabular-nums mono text-xs`)
- Fallback row visual: `border-l-2 border-yellow-500/70` on the row wrapper (D-08)
- Sort: Click column header for ascending/descending toggle on Timestamp, Task ID, or Fallback columns
- Pagination: `LoadMoreButton` at bottom — loads 25 more rows via cursor pagination
- Empty state: "No routing decisions recorded. Routing decisions are logged when gateway tasks are dispatched."
- Loading state: Skeleton rows (3 rows)

### GatewayTasksPanel
- Default 25 rows, most recent first (descending timestamp)
- Columns: Task ID (8-char mono), Provider badge, Status badge (PENDING/RUNNING/COMPLETED/FAILED — styled with status colors), Duration (ms, mono tabular-nums), Timestamp (relative)
- Row hover: `hover:bg-muted/50` consistent with EntityRow pattern
- FAILED rows: `text-red-400` on Status cell
- Sort: Timestamp (default), Provider, Status, Duration
- Filter: Provider dropdown (shadcn Select) — "All Providers" default
- Pagination: `LoadMoreButton` at bottom
- Empty state: "No gateway tasks recorded. Tasks appear here when routed through the CLI Gateway."
- Loading state: Skeleton rows (3 rows)

### ProviderComparisonChart (grouped FlexBarChart)
- 3 metric groups displayed side-by-side per provider: "Success Rate (%)", "Avg Latency (s)", "Task Count"
- Each provider gets its family color (see provider color table above)
- Providers with zero tasks in last 24h are hidden entirely (D-13)
- Chart height: 200px
- X-axis labels: provider display name (`PROVIDER_DISPLAY_NAMES`)
- Tooltip on hover: provider name + metric value
- Empty state: "No gateway task data in the last 24 hours."

### CostTrendChart (upgraded)
- Stacked FlexBarChart: each bar = one time bucket (hourly), each segment = one API-billed provider
- Segment colors: provider family color map above
- Tooltip on hover: time bucket label + per-provider cost breakdown
- Subscription providers ($0 cost) excluded from chart (D-17)
- Empty state: "No API cost data yet." (unchanged from existing)
