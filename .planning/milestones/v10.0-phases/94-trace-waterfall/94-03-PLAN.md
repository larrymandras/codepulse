<objective>
Build the new `TraceWaterfall` component (TRACE-02): a Gantt-styled, per-turn LLM call-chain visualization for a single session. It groups `llmMetrics` rows by `traceId` into collapsible turn groups (untraced legacy rows in one flat group last), renders each call as a timing bar with inline model/cost/cache label, a hover detail panel, and a `MetricCard` summary strip above. It reads `api.llm.sessionCalls` (from Plan 01) via Convex reactivity — no polling. It mirrors `GanttTimeline`'s structure but does NOT import it (D-09), and uses CSS theme tokens exclusively (no hardcoded hex).

Purpose: The visual core of the phase — where an operator reads a session's turns.
Output: `src/components/TraceWaterfall.tsx` + `src/components/TraceWaterfall.test.tsx`.
</objective>
