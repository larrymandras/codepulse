# Milestone v7.0 — Forge Integration Requirements (ARCHIVED)

**Shipped:** 2026-06-17 · **Status:** ✅ All 14 requirements satisfied

> **Goal:** Make Forge a first-class CodePulse module so all coding-agent work happens in one application, without moving Forge's execution engine off the local machine. Surface-Substrate bridge: state UP via bearer-authed httpActions, commands DOWN via a Convex command queue the daemon polls. Clerk-gated. *(Rejected: a cloud tab calling `http://localhost` directly — mixed-content blocked.)*

## Forge Foundation — Phase 78

- [x] **FI-01**: `forgeJobs` + `forgeWorkspaces` tables mirror the Forge job/workspace model (host-scoped, idempotent upsert keyed by `(hostId, forgeJobId)`)
- [x] **FI-02**: Local Forge emitter POSTs job state-change + periodic workspace sync to a bearer-authed `/forge-ingest` httpAction (401 bad key, 400 bad body)
- [x] **FI-03**: Read query API (`listJobs`, `getJob`, `listWorkspaces`) returns persisted Forge state in the UI's shape

## Forge UI Tab — Phase 79

- [x] **FI-04**: `/forge` route + CONSOLE nav (Flame) renders jobs/status/detail from `useQuery(api.forge.*)`, view-only
- [x] **FI-05**: StatusBadge / JobList / JobDetail / MetadataPanel ported ~1:1, re-skinned to CodePulse tokens, action controls stripped

## Command Bridge — Phase 80

- [x] **FI-06**: Convex `forgeCommands` queue the daemon long-polls; launch/stop delivered exactly once, execution status reflects back into `forgeJobs`
- [x] **FI-07**: Operator launches a new Forge job (ported NewJobModal) and stops a running job from `/forge`, round-tripping through the queue (launch 80-03, stop 80-04)
- [x] **FI-08**: Command-issuing mutations Clerk-gated — no unauthenticated launch/stop write path

## Live Log Streaming — Phase 81

- [x] **FI-09**: Bearer-authed `POST /forge-log-ingest` appends scrubbed lines to append-only `forgeLogChunks`; monotonic per-job `seq` makes re-delivery idempotent (`(hostId, forgeJobId, seq)` unique); 400/401/CORS
- [x] **FI-10**: Forge UI renders a per-job log pane from a reactive `forge.listJobLogs` query ordered by `seq`, updating live (Convex reactivity IS the stream — no SSE/WS)
- [x] **FI-11**: Scheduled retention sweep enforces a 7-day TTL AND a per-job byte/chunk cap (drop-oldest), verified by a cron/cleanup test

## Files + Artifact Preview + Hardening — Phase 82

- [x] **FI-12**: Operator browses a job's workspace files and previews artifacts in `/forge` (ported FileBrowser / ArtifactPreview)
- [x] **FI-13**: Artifact/file content reachable from the cloud UI via the Convex bounded-ingest bridge — daemon → `/forge-file-ingest` → `forgeFiles`/`forgeArtifacts` (+ File Storage for images) → cloud UI; **no mixed-content `http://localhost`**, tunnel/local-https rejected. Verified live 2026-06-17 (listing bridge end-to-end; preview bytes via forge `dbfad91` ACL grant).
- [x] **FI-14**: End-to-end Clerk gating across the Forge surface + FI-14 preview security invariants (sandboxed `data:` iframe, no `allow-same-origin`, no `dangerouslySetInnerHTML`); full launch→run→logs→artifacts path auth-correct
