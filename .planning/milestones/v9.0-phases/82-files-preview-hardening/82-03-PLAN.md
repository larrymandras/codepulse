## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Convex artifact bytes → preview iframe | Agent-generated HTML/text rendered in the browser; must be sandboxed |
| Convex File Storage URL → <img> | Unauthenticated blob URL rendered as image (D-02a) |
| File path/name → DOM | Potentially attacker-controlled filename rendered in the list |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-82-10 | Tampering / Info Disclosure | XSS via agent HTML in preview | mitigate | iframe `sandbox="allow-scripts"` ONLY (no allow-same-origin); src is a `data:` URI (null origin); Source is `<pre>{textContent}</pre>` React text node; test asserts sandbox value + escaped source |
| T-82-11 | Tampering | XSS via crafted filename | mitigate | filenames/paths rendered as React text nodes `{entry.path}`; no innerHTML; test asserts escaped rendering of a script-tag filename |
| T-82-04 | Info Disclosure | unauthenticated image URL | accept | D-02a: Convex File Storage URL is unguessable/permanent, image content non-executable, behind AuthGuard'd `/forge`; passed only to `<img src>` |
| T-82-12 | Availability | preview render crash cascades | mitigate | Files tab body wrapped in `<SectionErrorBoundary name="Files">`; a render failure does not take down Details/Logs (Req 11) |
| T-82-13 | Info Disclosure | bearer key in browser bundle | mitigate | No ingest env var imported in `src/`; audit `grep -r FORGE_INGEST_API_KEY src/` returns zero (cross-plan audit; this plan adds no such reference) |
| T-82-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages (82-RESEARCH: ExternalLink/FolderOpen are imports from already-installed lucide-react; zero installs) |
</threat_model>
