War Room Launch Flow

Two paths to launch — from Teams page or ad-hoc

Path 1: From Team Preset

1. Click "Launch War Room" on any team card
2. Optional: add/remove agents for this session
3. Set topic/agenda (optional)
4. Launch → POST /api/war-room
{ participants: ["astridr", "hervor", "ragnhildr", "sigrun"],
  topic: "Security audit review",
  teamPresetId: "technical-review" }

Path 2: Ad-hoc from Roster

1. Select agents from Roster (checkbox multi-select)
2. Bulk action bar appears: "Launch War Room with 3 agents"
3. Set topic/agenda (optional)
4. Optional: "Save as Team" before launching
5. Launch → POST /api/war-room
{ participants: ["astridr", "gondul", "skuld"],
  topic: "Content sprint planning" }

NORSE_AGENT_NAMES = ["astridr", "hervor", "freya", "gondul", "ragnhildr"]
for agent_name in NORSE_AGENT_NAMES:
  await lk.agent_dispatch.create_dispatch(...)
# AFTER (dispatcher.py)
DEFAULT_PARTICIPANTS = ["astridr", "hervor", "freya", "gondul", "ragnhildr"]

async def create_war_room(participants: list[str] | None = None, ...):
  agent_ids = participants or DEFAULT_PARTICIPANTS
  for agent_id in agent_ids:
    config = registry.get(agent_id)
    if config is None:
      raise ValueError(f"Agent {agent_id} not in registry")
    await dispatch_agent(room_name, agent_id, config)
# NEW: DynamicWarRoomAgent
class DynamicWarRoomAgent(NorseAgent):
  """Loads personality from AgentTypeConfig at dispatch time.
   Works for any agent in the registry, not just original 5."""

  def __init__(self, config: AgentTypeConfig):
    self.persona_id = config.id
    self.system_prompt = load_soul_variant(config)
    self.display_name = config.name
⚠️ Original 5 Norse agents keep their dedicated subclasses. DynamicWarRoomAgent is used for any newly onboarded agent. Existing war room behavior unchanged when no participants specified.

Data Model: teamPresets (Convex)

Convex table for team presets
teamPresets: defineTable({
  name: v.string(),               // "Technical Review"
  description: v.optional(v.string()),// "Architecture and code decisions"
  agentIds: v.array(v.string()),    // ["astridr", "hervor", "ragnhildr"]
  createdAt: v.number(),
  updatedAt: v.number(),
  createdBy: v.optional(v.string()),
  lastUsedAt: v.optional(v.number()),// track which teams are active
  warRoomCount: v.optional(v.number()),// how many times launched
})
